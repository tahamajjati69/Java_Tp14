# Projet Java CRUD avec DAO Générique

## 1. Description du projet
Ce projet implémente un système **CRUD (Create, Read, Update, Delete)** en Java avec une **architecture en couches**.  
Il gère des **Profils** et des **Utilisateurs** avec persistance **en mémoire** via un DAO générique (`ListDao<T>`).  
Le projet illustre l’utilisation des interfaces, des classes génériques et la séparation des responsabilités.
